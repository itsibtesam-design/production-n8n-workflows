## ⚡ How to Import & Deploy These Workflows

Every automation in this repository is production-ready and can be deployed to your local or cloud n8n instance in under 60 seconds:

1.  Navigate to the specific folder above and click on the desired `.json` file.
2.  Click the **Raw** button on the top right of the file view to expose the clean JSON data.
3.  Press `Ctrl + A` (or `Cmd + A`) followed by `Ctrl + C` (or `Cmd + C`) to copy the entire payload.
4.  Open a blank canvas in your **n8n workspace**.
5.  Simply click anywhere on the canvas and press `Ctrl + V` (or `Cmd + V`). The entire node architecture, including mapping configurations and logic branches, will populate instantly.

> ⚠️ **Note on Configuration:** To make these workflows operational, ensure you configure your own environment variables, webhooks, and API credentials within the respective node settings.
